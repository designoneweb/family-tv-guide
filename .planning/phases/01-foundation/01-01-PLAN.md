<task type="auto">
  <name>Task 2: Configure shadcn/ui with dark theme</name>
  <files>web/components.json, web/app/globals.css, web/lib/utils.ts, web/tailwind.config.ts</files>
  <action>
1. Run `npx shadcn@latest init` in web/ directory:
   - Style: Default
   - Base color: Slate (good for dark mode)
   - CSS variables: Yes
   - tailwind.config location: tailwind.config.ts
   - Components alias: @/components
   - Utils alias: @/lib/utils

2. Update web/app/globals.css to force dark mode:
   - Add `@layer base { html { @apply dark; } }` after the shadcn CSS variables
   - Ensure the dark theme variables are using deep slate/zinc tones for art-forward look

3. Update web/app/layout.tsx:
   - Add `className="dark"` to html element
   - Set appropriate metadata (title: "Family TV Guide", description)
   - Use Inter or system font stack

4. Install TanStack Query for later use: `npm install @tanstack/react-query`
  </action>
  <verify>npm run dev shows dark-themed page, no console errors, components.json exists</verify>
  <done>shadcn/ui initialized with dark mode forced on, TanStack Query installed</done>
</task>
