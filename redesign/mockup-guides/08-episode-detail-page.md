# Page 08: Episode Detail Page

## Page Purpose
The deepest level of detail—a single episode's full information including synopsis, guest cast, and viewing actions. This feels like reading the back of a DVD case for that specific episode, or a detailed TV guide entry.

---

## Layout Structure

### Overall Composition
- Immersive hero with episode still
- Episode metadata section
- Synopsis/description area
- Guest cast gallery
- Navigation to adjacent episodes
- Feeling: A detailed episode guide entry

---

## Hero Section

### Container
- **Height**: 50vh (desktop), 40vh (mobile)
- **Position**: Relative
- **Overflow**: Hidden

### Background Image
- **Source**: TMDB still_path for this episode
- **Fallback**: Show backdrop if no still
- **Treatment**:
  - Cover entire container
  - Desaturate to 85%
  - Subtle Ken Burns: Slow zoom in (15s cycle)
  - Heavy gradient from bottom: Deep Night 100% → transparent at 50%

## Responsive Behavior

### Desktop (1200px+)
- Hero: 50vh
- Synopsis: Centered, max-width 800px
- Guest stars: Grid wrap
- Navigation: Side-by-side cards

### Tablet (768px - 1199px)
- Hero: 45vh
- Content: Full width with padding
- Guest stars: Horizontal scroll
- Navigation: Stacked

### Mobile (< 768px)
- Hero: 40vh
- Breadcrumb: Simplified (just back arrow + show name)
- Action bar: Fixed bottom, compact
- Synopsis: Full width
- Guest stars: Horizontal scroll, larger cards
- Navigation: Vertical stack, full-width cards

---

## Animations

### Page Load
1. Hero image fades in
2. Breadcrumb slides in from left
3. Content staggered fade up (badges, title, meta)
4. Action bar slides up
5. Synopsis section fades in
6. Guest stars stagger in (if visible)

### Interactions
- Mark Watched:
  1. Button shows checkmark animation
  2. "WATCHED" badge animates in from right
  3. Hero tints slightly
  4. Next episode navigation highlights/pulses
- Navigation cards: Hover scales thumbnail
- Guest star cards: Smooth scale + shadow

### Transitions
- To next/prev episode: Slide transition (content slides in direction of navigation)
- To actor page: Zoom transition centered on clicked photo

---

## Accessibility Notes

- Breadcrumb navigation is semantically correct
- Episode navigation has proper prev/next relationship
- Mark Watched state announced to screen readers
- Guest star photos have alt text
- Synopsis is properly structured with headings
- AI summary marked as supplementary content
- Keyboard navigation through guest star grid
- Episode navigation available via keyboard
